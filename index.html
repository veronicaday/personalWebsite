C++ Search Engine

Search engine wrote entirely from scratch in C++
https://github.com/vday14/cplusplus-search-engine

Download our project report

In my Winter semester, I took an experimental class called System Design in C++. The class was taught by Professor Hamilton, who's professional accomplishments include implementing the ranking engine for Bing. Our team, along with 5 other teams in the class, worked to design and build a search engine without using external libraries (we even built our own data structures). We implemented the Crawler, Parser, Index, Constraint Solver, Stream Readers, Query Language, Ranker, and Front End.

The best part about this project was learning how to break a rather large and daunting project into smaller pieces. We also learned how to effectively fit many pieces of a project together, even if they are being development in parallel and by different people. Concretely, we learned about how file handling works, how to host a local server, how to use a hash table of disk, and how to think about storing large amounts of data with quick access (without using a database language like MySQL or ElasticSearch). by far, this has been my favorite class I've taken.

Gender Bias Detector


Predict a professor's gender based on their RateMyProfessor profile
https://github.com/vday14/gendered-rmp

Download our project report



We used Ben Schmidt's findings on gender bias in RateMyProfessor reviews (http://benschmidt.org/profGender) as our inspiration for this project. This project was part of my group's final project for our Information Retrieval class (EECS 486).
We attempted to determined the genders of University of Michigan professors on RateMyProfessor through users comments and ratings. From this information, we found correlations between certain words and the gender of a professor.
Our project was accomplished by crawling RateMyProfessor pages and collecting comments and tags on each professors page. We compared the findings from three different models -- a vector space model with TF-IDF weighting, the Nearest Neighbor (K of 1, 11, and 25) and Rocchio text classification methods to determine the gender of a professor. I personally worked on the web crawler, which used selenium and beautiful soup to collect the data from RMP.

Code Linter

Simplify writing clean code

During my summer internship with Trove, I developed a code linter that runs automatically before code is pushed into production. It recognizes and fixes stylistic errors according to custom guidelines (sniffs), and regulates proper code architecture.

For example, the linter will detect misaligned brackets, incorrectly named variables, and use statements that are unused or out of order. The code fixer will then run over the files and "fix" all detected errors automatically. This standardizes our code reviews, ensures correct code architecture, and helps developers quickly fix errors and avoid writing spaghetti code.

AI Poetry & Music Generator

Use NLP techniques to generate haikus and melodies

For our EECS 183 final project, my team utilized N-gram language modeling (NLP) to train our program to create songs based on lyrics from Beatles, combined with a syllable parser to generate Haiku and rhyming poetry. Robotic Acetabular Detection

Improve detection of bone during robotic hip replacement surgery

During my summer internship at Stryker, I acted as a liaison between the R&D and product strategy teams for our robotic division. My project was to evaluate the reason why surgeons were having a difficult time adapting to our new robotic technology. After viewing multiple surgical videos of users interacting with the technology, it became apparent that the frequency of errors was frustrating surgeons - even though using the robotic lead to faster surgery and quicker healing time for patients.

I designed a metric for evaluating the efficiency of how the robot's haptics, camera set-up, and instrument design interfered with the usability. I collaborated with a team of UI/UX engineers on potential improvements for the next generation surgical robot.
